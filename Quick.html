Quick Instructions

Step 1: Get and install LAMMPS with FitSNAP

Here are the steps to setup current LAMMPS with ML fitting capability. You can copy and paste this entire block in your terminal if you please:

# remove current python lammps interface

rm -rf /srv/conda/envs/notebook/lib/python3.8/site-packages/lammps*

# get dependencies

python -m pip install numpy torch scipy virtualenv psutil pandas tabulate mpi4py Cython

# get and install lammps

cd $HOME
git clone https://github.com/lammps/lammps
mkdir lammps/build-fitsnap
cd lammps/build-fitsnap
cmake ../cmake -DLAMMPS_EXCEPTIONS=yes \
               -DBUILD_SHARED_LIBS=yes \
               -DMLIAP_ENABLE_PYTHON=yes \
               -DPKG_PYTHON=yes \
               -DPKG_ML-SNAP=yes \
               -DPKG_ML-IAP=yes \
               -DPKG_ML-PACE=yes \
               -DPKG_SPIN=yes \
               -DPYTHON_EXECUTABLE:FILEPATH=`which python`
make
make install-python

# get FitSNAP

cd $HOME
git clone https://github.com/FitSNAP/FitSNAP

Now we have lammps and FitSNAP directories.

Put these environment variables in your ~/.bashrc:

SITE_PACKAGES_DIR=`python -c "import site; print(site.getsitepackages()[0])"`
LAMMPS_DIR=$HOME/lammps
FITSNAP_DIR=$HOME/FitSNAP

export PYTHONPATH=${SITE_PACKAGES_DIR}:$PYTHONPATH # So that ML-IAP package can find torch for MD
export PYTHONPATH=${FITSNAP_DIR}:$PYTHONPATH # So that you can run FitSNAP as executable
export LD_LIBRARY_PATH=${HOME}/lammps/build-fitsnap

Activate your ~/.bashrc:

source ~/.bashrc

Step 2: Fit neural network potential

We’ll do a simple example with tantalum; this dataset has crystals, surfaces, and liquids:

cd ~/FitSNAP/examples/Ta_PyTorch_NN
python -m fitsnap3 Ta-example.in --overwrite # overwrite will overwrite existing fits

This creates a FitTorch_Pytorch.pt file that is used by LAMMPS for MD.

Step 3: Run MD with neural network potential

Run MD with:

cd $HOME/FitSNAP/examples/Ta_PyTorch_NN/MD
~/lammps/build-fitsnap/lmp < in.run
